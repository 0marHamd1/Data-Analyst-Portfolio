# Data-Analyst-Portfolio

# Project Overview:

This project analyzes sales performance to identify key revenue drivers and profitability issues.

I built a Sales Analysis Dashboard using Power BI to analyze business performance and identify key insights.

🔍 Key Insights:
Technology is the top-performing category in both sales and profit
Furniture has high sales but low profitability due to heavy discounting
Central region shows strong sales but weak profit margins
Some top-selling products are actually generating losses

💡 Recommendations:
Focus on high-margin categories
Optimize discount strategies
Revaluate loss-making products

📊 Tools Used:
Excel
Power BI
